Información

  • Mínimo: Se acepta que esta CPU puede causar un cuello de botella ligero o moderado en algunos juegos, pero la mayoría funcionará sin problema.
  • Recomendado: Es el equilibrio calidad-precio perfecto: la inmensa mayoría de juegos no tendrá cuello de botella y el precio es acorde al de la gráfica.
  • Máximo: Es la CPU de gama más alta que tiene sentido para esta gráfica (no se toma en cuenta la calidad-precio, pero sí evitar usar una CPU todavía más alta si una menos potente ya va sobrada para la gráfica seleccionada).
